# Support

If you need help with provii-demos, the following resources are available.

## Documentation

Full integration guides, API references, and walkthroughs live at [docs.provii.app](https://docs.provii.app).

## Playground

Try the age verification flow interactively at [playground.provii.app](https://playground.provii.app). The playground lets you test credential issuance and verification without writing any code.

## Bug Reports

Found a bug? Open an issue using the bug report template in this repository. Include steps to reproduce, the demo backend language you were using, and any relevant logs.

## Security Issues

Do not open a public issue for security vulnerabilities. Follow the process described in [SECURITY.md](SECURITY.md) to report them through our security advisory channel.
